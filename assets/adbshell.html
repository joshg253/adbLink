Android only

The "ADB Shell" button opens a telnet-like session on the selected device.

adbLink installs and uses Busybox, a utility that adds dozens of unix utility programs that enhance Android. To add busybox to your path for command-line use, the Path to clipboard menu function is provided.

If you have questions or comments, please email jocala@jocala.com or visit one of the forums linked below. adbLink produces a log file, adblink.log, with each run. Include it in your email or post if you're having problems.
