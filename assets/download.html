The "Download Kodi" menu function opens the kodi.tv download page in your system's browser.

Scroll to the "Current Release" section: At this time the download is: Jarvis 16.1 Android ARM", and the filename is: kodi-16.1-Jarvis-armeabi-v7a.apk

This download will change depending on the current release of Kodi.

The "Download SPMC" menu function will open the main page of the SPMC media center. Download the current APK file, not the source code.
