Android Kodi, by default, stores its data on partition /sdcard/. Unfortunately, on some devices /sdcard/ has limited space, so the "Move Kodi Data" allows you to move your entire Kodi data folder to your device's external storage. Please note that Kodi MUST have been run for the first time before using this dialog.

The dialog presents a list of your currently mounted external storage and two radio buttons:

  • Move data from /sdcard to selection
  • Move data from selection to /sdcard/

Select the correct partition, click the radio button of your choice and press "OK".

The Move data choices move Kodi's entire data folder (.kodi) to the external device selected, or back to the default /sdcard/ partition.

At start, Kodi looks for a text file "/sdcard/xbmc_env.properties" that tell it where its data files are located. adbLink creates this file when it moves the data directory and deletes the file when you move the directory back to its default.

Moving data is not allowed if Kodi is running.

If you have questions or comments, please email jocala@jocala.com or visit one of the forums linked below. adbLink produces a log file, adblink.log, with each run. Include it in your email or post if you're having problems.
