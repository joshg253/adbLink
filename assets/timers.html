At times, when an Android device enters sleep mode or when the device's screensaver starts it can have a disruptive effect on Kodi. This dialog allows you to adjust the timers for sleep and screensaver, even turning the services off so they don't start.

The dialog shows two sets of controls, for the sleep timer and for the screensaver timer.

The radio buttons offer choices to turn the timer off or return it to the default state.

The label below the radio buttons show the timer's current status.

The text entry field allows you to set a custom value in milliseconds for the timer.

The default values (in milliseconds) for the Amazon Fire TV are:

default sleep: 1200000
sleep off: 0

default screensaver: 300000
screensaver off: 2147460000
